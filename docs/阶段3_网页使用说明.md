# VMS 阶段 3 网页使用说明

## 一、启动系统

打开 PowerShell，运行：

```powershell
powershell -ExecutionPolicy Bypass -File D:\VMS_V1\backend\run_backend.ps1
```

保持窗口不要关闭。浏览器访问：

```text
http://127.0.0.1:8000/
```

如果之前已经启动过阶段 2 服务，请先在旧 PowerShell 窗口按 `Ctrl+C` 停止服务，再重新运行启动命令。

## 二、初始化演示数据

左侧点击“样例导入”，依次点击：

1. “导入资产样例”
2. “导入漏洞样例”

漏洞样例月份默认填写 `2026-05`。首次导入后，工作台应显示：

| 项目 | 数量 |
|---|---:|
| 漏洞总数 | 13 |
| 青藤云 | 5 |
| 阿里云 | 7 |
| 绿盟 | 1 |
| 导入异常 | 6 |

异常来自绿盟索引中存在、但样例目录未提供对应 `{IP}.xls` 报告的主机。这是预期结果。

## 三、页面说明

| 页面 | 当前功能 |
|---|---|
| 工作台 | 查看当月漏洞总数、未修复数、已处置数、涉及项目数、项目待下线数、导入异常数，以及扫描器和状态分布 |
| 样例导入 | 从项目样例文件生成开发演示数据 |
| 资产管理 | 查看 IP、组织、项目、工作空间、负责人映射 |
| 漏洞清单 | 按扫描器、处置状态、项目筛选，修改状态和备注 |
| 异常处理 | 查看导入失败或报告缺失的记录 |

顶部月份影响工作台、漏洞清单和异常处理页面。修改月份后点击“刷新当前页面”。
