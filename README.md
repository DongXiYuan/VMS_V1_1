# VMS_V1_1

`VMS_V1_1` 是面向漏洞推动场景整理后的独立项目版本，对应当前 `V1.1` 交付内容。

项目核心目标仍然围绕三类业务能力：

1. 将不同扫描器输出的 Excel 或 ZIP 文件转换为统一标准漏洞清单
2. 基于标准漏洞清单进行查询、统计、月度对比和异常分析
3. 通过页面动态维护处置状态、备注，并生成按项目拆分的导出结果

## V1.1 主要能力

相较基础版本，`V1.1` 重点增强了以下能力：

- 批量修改漏洞处置状态与备注
- 批量软删除漏洞记录
- 未匹配资产导出与去重导出
- 异常资产批量重匹配
- 系统漏洞总数 Top10
- 漏洞名称出现次数 Top10
- 复发漏洞识别、高亮、筛选
- 工作台点击钻取具体漏洞明细

## 快速启动

在 Windows PowerShell 中执行：

```powershell
cd D:\VMS_V1_1
pip install -r .\backend\requirements.txt
powershell -ExecutionPolicy Bypass -File .\backend\run_backend.ps1
```

浏览器访问：

```text
http://127.0.0.1:8000/
```

接口文档：

```text
http://127.0.0.1:8000/docs
```
